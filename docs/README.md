# Echo

Минималистичный агрегатор для тех, кто ценит чистоту и объективность.
Он группирует статьи из разных СМИ по одной теме в единый сюжет, позволяя
увидеть разные точки зрения на одно событие без лишних повторов.

**Стек:** Натив с прямым использованием Android SDK.

## Сборка и установка

Запустим нужный нам эмулятор. Если вы хотите использовать физическое
устройство, то следующие две команды нужно пропустить:

```txt
./emulator -list-avds
```

```txt
./emulator -avd <emulator_name>
```

Затем соберите и установите на него:

```txt
./gradlew installDevDebug
```

## Моки

Все API моки находятся в пакете `api` внутри модуля `:lib`.

## Запуск ключевого сценария

```txt
./adb shell monkey -p ru.tsu.echoSample.app.dev.debug -c android.intent.category.LAUNCHER 1
```

После выполнения этой команды, вы сразу увидите **ключевой
сценарий «Мониторинг популярности мнений».**
